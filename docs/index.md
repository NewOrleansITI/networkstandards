# City of New Orleans Network Standards

Welcome to the official network infrastructure standards documentation for the City of New Orleans Department of Information Technology and Innovation.

## Purpose

This documentation provides standardized specifications and procedures for deploying and maintaining network infrastructure across City of New Orleans facilities. Following these standards ensures consistency, reliability, and security across all city networks.
