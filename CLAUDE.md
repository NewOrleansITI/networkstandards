## Repository Context

This is the City of New Orleans Network Standards repository containing vendor-neutral networking documentation for municipal infrastructure.

## Key Policies

- **WiFi 7 Mandatory** — All new AP deployments must use 802.11be (2026)
- **4 SSID Maximum** — Hard limit per access point for performance
- **OWE Required** — Guest networks must use Enhanced Open encryption

## Documentation Standards

- Follow existing document structure and formatting
- Reference only public standards bodies (IEEE, ANSI, TIA, IETF, ISO, NIST)
- No vendor-specific recommendations
- Priority order: security > reliability > cost-effectiveness > features
